# Alya Pack – Minecraft Resource Pack (Fan-Made)

**Alya Pack** is a fan-made Minecraft resource pack inspired by Alya and other characters from the anime *“Alya Sometimes Hides Her Feelings in Russian.”*  

This pack replaces selected Minecraft textures with Alya-themed visuals, focusing on a clean, anime-style aesthetic that blends naturally into vanilla gameplay. It adds subtle character flair without breaking immersion or readability.

## Features
- Alya-inspired textures for specific items and UI elements
- Clean, anime-style artwork that fits Minecraft’s look
- Lightweight and performance-friendly
- Compatible with most other resource packs

## Why Download?
If you enjoy anime-themed customization and want to personalize your Minecraft experience without overwhelming visual noise, this pack adds character and personality while keeping the game familiar and playable.

## How to Download & Install
1. Click the **Download** button to get the `.zip` file.  
2. Open Minecraft → **Options → Resource Packs → Open Pack Folder**.  
3. Move the downloaded `.zip` file into the folder.  
4. Back in Minecraft, activate the pack and enjoy!
